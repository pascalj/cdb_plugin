# Demo for Compilation Database Plugins

Enhance a compilation database (`compile_commands.json`) for headers and unknown files by using an include graph. This
is a demo project and not meant for production.

## Building

The dependencies are clang-dev 17 and CMake.

```
mkdir build
cd build
cmake ..
```
